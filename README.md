# Loan Default Prediction App

A Machine Learning project that predicts whether a customer is likely to default on a loan based on financial and loan-related information.

## 📌 Project Overview
This project was built to understand the complete ML workflow including:
- Data preprocessing
- Outlier handling
- Feature engineering
- Model training & evaluation
- Model deployment using Streamlit

The application allows users to enter customer and loan details and get a real-time loan default prediction.

---

## 🚀 Features
- Predicts loan default risk
- Real-time prediction using Streamlit
- Displays default probability
- Interactive user interface
- Trained using multiple ML algorithms

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Joblib

---

## 📊 Machine Learning Workflow
1. Data Cleaning & Preprocessing  
2. Outlier Detection & Handling  
3. Feature Engineering  
4. Encoding Categorical Variables  
5. Model Training & Evaluation  
6. Model Comparison  
7. Deployment using Streamlit  

---

## 📈 Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

Among all models, XGBoost performed best based on precision, recall, F1-score, and overall prediction performance.

---

## 🎯 Key Insights
- Interest rate and loan burden (`loan_percent_income`) were important factors affecting loan default risk.
- Higher interest rates showed a higher probability of default in the dataset.
